Using HTML5 table tags

This is a simple table with header using caption, tr, th, and td tags:

My simple table
those are headers
those are normal cells

The table uses the border attribute which defines, in pixels, the size of the frame surrounding the table. If set to 0, it implies that the frame attribute is set to void. This attribute has been deprecated and the <table> element should be styled using CSS. So let's create the following tables without this attribute.

This is a table with thead, tfoot, and tbody tags:

Header content 1 Header content 2
Footer content 1 Footer content 2
Body content 1 Body content 2

The thead tag define a set of rows defining the head of the columns of the table. The tfoot defines a set of rows summarizing the columns of the table. And the tbody tag defines one or more <tr> element data-rows to be the body of its parent table element.

The colgroup tag defines a group of columns within a table.

The colgroup tag defines a column within a table and is used for defining common semantics on all common cells.
